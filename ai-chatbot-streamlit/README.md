# 🤖 GenAI Chatbot (Streamlit)

This is an interactive AI chatbot built using Python, Streamlit, LangChain, and OpenAI APIs.

## 🚀 Features
- Real-time chatbot
- Session-based memory
- Dynamic model selection
- Prompt engineering

## 🧠 How It Works

- User enters a query
- Prompt is structured using LangChain
- LLM generates response via OpenAI API
- Chat history is stored using Streamlit session state

## 🛠 Tech Stack
- Python
- Streamlit
- LangChain
- OpenAI API

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
